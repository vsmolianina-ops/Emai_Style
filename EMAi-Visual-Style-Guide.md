# EMAi Visual Style Guide
> Дизайн-система для сайту emai.group — використовуй цей документ для переробки презентацій, матеріалів, слайдів у стилі бренду.

---

## 1. Кольорова палітра

### Основні кольори

| Назва | HEX | Використання |
|---|---|---|
| **Cream** | `#F7F4EB` | Основний фон сторінки, секції з кремовим тлом |
| **Cream Soft** | `#FBF9F3` | Світліший варіант для вкладених блоків |
| **Cream Deep** | `#EFE9D9` | Розділювачі, hover-стани на кремовому |
| **Accent** | `#D69043` | Основний акцент — ціни, іконки, посилання, активні елементи |
| **Accent Soft** | `#E8B074` | Світліший акцент для hover, другорядних елементів |
| **Accent Deep** | `#B87530` | Темніший акцент, натиснутий стан |
| **Ink** | `#2D3142` | Заголовки H1–H3, найважливіший текст |
| **Dark** | `#5E6472` | Основний текст тіла, підзаголовки |
| **Dark Soft** | `#7A8090` | Підписи, мітки, вторинний текст |
| **Line** | `#E2DDD0` | Розділювальні лінії, бордери карток |
| **White** | `#FFFFFF` | Фон секцій, картки, Nav |

### Як поєднувати кольори

```
Cream фон   → Ink заголовки + Dark текст + Accent акценти
White фон   → Ink заголовки + Dark текст + Accent акценти
Dark фон    → White текст (кнопка Consult)
Accent фон  → Cream або White текст
```

### Для слайдів — рекомендовані фони

- **Тло слайду:** `#F7F4EB` (Cream) або `#FFFFFF` (White)
- **Акцентний слайд (title slide, розділювач):** `#2D3142` (Ink) з білим текстом
- **Ніколи:** яскраво-синій, зелений, червоний — не в палітрі бренду

## 2. Типографіка

### Шрифти

| Роль | Шрифт | Де застосовується |
|---|---|---|
| **Display** | Gantari Variable | Всі заголовки H1–H4, великий текст |
| **Body** | Geist Variable | Основний текст параграфів |
| **UI** | Inter Variable | Кнопки, мітки, навігація, підписи |

> **Google Fonts альтернатива для презентацій:**
> Якщо Gantari недоступний — використовуй **DM Sans** або **Plus Jakarta Sans**.
> Якщо Geist/Inter недоступний — використовуй **Inter** (є у Google Fonts).

### Розміри та ваги

| Елемент | Розмір | Вага | Шрифт |
|---|---|---|---|
| Hero H1 | 72–84px (десктоп) / 44px (моб) | Bold 700 | Gantari |
| Section H2 | 48–60px | Medium 500 | Gantari |
| Card H3 | 22–26px | Medium 500 | Gantari |
| Body large | 18–20px | Regular 400 | Geist |
| Body | 16px | Regular 400 | Geist |
| UI / Label | 13–14px | Medium 500 | Inter |
| Price large | 48px | Bold 700 | Inter |
| Price small | 30px | Bold 700 | Inter |

### Правила типографіки

- **Letter-spacing заголовків:** `-0.02em` (трекінг злегка стиснутий)
- **Line-height заголовків:** `1.1–1.15`
- **Line-height тексту:** `1.6–1.7` (relaxed)
- **Text-wrap:** `balance` для заголовків, `pretty` для параграфів

---

## 3. Відступи та сітка

### Контейнер контенту
- **Максимальна ширина:** `1280px`
- **Горизонтальний padding:** `24px` (моб) / `40px` (планшет+)

### Вертикальні відступи секцій
| Breakpoint | Padding top/bottom |
|---|---|
| Mobile | `64px` |
| Tablet (md) | `80px` |
| Desktop (lg) | `96px` |

### Внутрішні відступи карток
- **Padding картки:** `28px` (моб) / `36px` (десктоп)
- **Gap між картками:** `16px` (моб) / `24px` (десктоп)

### Сітка колонок

| Контент | Колонки |
|---|---|
| Текст + зображення | `1fr 1fr` (50/50) |
| B2B Features | `repeat(2, 1fr)` → `repeat(4, 1fr)` |
| Pricing | `repeat(2, 1fr)` |
| Privacy/Features список | `1fr` (одна широка колонка) |

## 4. Компоненти

### Кнопка CTA (основна)
```
Фон:      #2D3142 (Ink)
Текст:    #FFFFFF, 14px, Medium, Inter
Hover:    трохи темніший (opacity 90%)
Padding:  12px 24px
Radius:   12px (rounded-xl)
Height:   42px
```

### Кнопка вторинна / outline
```
Фон:      прозорий
Border:   1px solid #E2DDD0
Текст:    #5E6472, 14px, Medium, Inter
Radius:   12px
```

### Картка
```
Фон:      #FFFFFF
Border:   1px solid #E2DDD0
Radius:   24px (rounded-3xl)
Shadow:   0 8px 32px rgba(45, 49, 66, 0.08)
Padding:  28–36px
```

### Акцентний кружечок (іконка)
```
Розмір:   64×64px
Фон:      #D69043 (Accent)
Колір:    #F7F4EB (Cream)
Radius:   50% (circle)
```

### Ціна
```
Велике число:   48px, Bold, Inter, #D69043
Маленький текст (/device, /month):   14px, Regular, #7A8090
Знак +:   30px, Bold, Inter, #D69043
```

### Розділювальна лінія
```
Колір:    #E2DDD0 (Line)
Товщина:  1px
Opacity:  30–50%
```

## 5. Іконографія

- **Стиль:** лінійні (stroke), не заповнені
- **Товщина лінії:** 1.5–1.75px
- **Колір:** `#5E6472` (Dark) або `#D69043` (Accent) для акцентних
- **Розмір у картках:** 48–54px висота, auto ширина
- **Checkmark:** stroke, `#D69043`, товщина 1.75px

---

## 6. Фотографії та зображення

- **Стиль:** тепле природне освітлення, реальні люди (літні дорослі, родини)
- **Настрій:** спокійний, довірливий, домашній
- **Уникати:** стокові кліше, надто яскраві кольори, холодне синє освітлення
- **Overlay на відео:** `rgba(0,0,0,0.30)` градієнт знизу для читабельності тексту
- **Aspect ratio карток:** `1:1` (квадратні) або `4:3`

---

## 7. Паттерни лейауту

### Чергування текст ↔ зображення (Families секція)
```
Slide 1: [Текст зліва] [Фото справа]
Slide 2: [Фото зліва]  [Текст справа]
```

### Full-bleed секція
- Фон розтягується на всю ширину браузера
- Текст і контент обмежені контейнером 1280px

### Hero
- Відео / велике фото на весь екран
- Текст у лівому нижньому куті
- Легкий темний градієнт знизу

---

## 8. Брендові правила

### Логотип
- **Основний:** EMAi wordmark (кольоровий: `#5E6273` літери + `#D69043` крапка над "i")
- **На темному фоні:** білий варіант
- **Мінімальний розмір:** 80px ширина
- **Підпис:** "caring by tech" — `#D69043`, Inter Medium, 12–13px

### Назва бренду
- Завжди: **EMAi** (не EMAI, не Emai, не emai)
- "i" завжди рядкова, решта великі

### Tone of voice
- Спокійний, турботливий, технологічний але людяний
- Не агресивний маркетинг
- Простота і зрозумілість (аудиторія — літні люди та їх родини)
